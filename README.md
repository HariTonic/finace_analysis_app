# Finance Management App

A Flutter app for managing personal finances with Hive local storage.

## Features

- Dashboard with balance, income, expense summary
- Add income and expense transactions
- View all transactions with filters
- Reports with monthly summaries
- Backup and restore data

## Setup

1. Install Flutter SDK
2. Run `flutter pub get`
3. Run `flutter pub run build_runner build` to generate Hive adapters
4. Run `flutter run` to start the app

## Project Structure

- lib/main.dart: App entry point
- lib/screens/: UI screens
- lib/models/: Data models
